## 그룹 A: Brain + Mind 모듈

| | |
|---|---|
| **구성원** | 석혜원 (Brain 모듈 리드, E4·E10), 안우영 (Mind 모듈 리드, E2·E3), 고성룡, 김향숙 |
| **진행자** | 석혜원, 안우영 (공동 진행) |

### 토론 질문

> **"PBL 전환 시 실험·실습 과목의 실습 시간을 어떻게 확보할 것인가?"**

### 배경 정보

- 국내외 메타분석에서 PBL/CBL이 심리학 교육에 효과적임이 반복 확인되고 있으며, 최적 조건은 **4~5명 소그룹, 한 학기(9~18주), 50명 이하 학급**으로 보고됨
- Brain 모듈 과목(fMRI, EEG, PsychoPy 실습 등)은 **장비 기반 실습 비중이 높아** 기존 강의 시간 외에 별도 실습 시간이 필수적임
- Mind 모듈 과목(인지실험 설계·실행)도 **실험 장비 및 참가자 운영 시간**이 상당하여 PBL 프로젝트와의 시간 배분이 핵심 쟁점임
- 본 토론은 PBL 도입 여부가 아니라, **실습 시간 확보를 위한 실질적 운영 방안**을 논의하는 자리임

## 그룹 B: Behavior 모듈

| | |
|---|---|
| **구성원** | 최인철 (Behavior 모듈 리드, E5-E8), 김가원, 최진영, 이해연 |
| **진행자** | 최인철 |

### 토론 질문

> **"대규모 수업(60명 이상)에서 PBL 소그룹(4-5명) 운영이 현실적인가?"**

### 배경 정보

- 국내 대학 PBL 연구에서도 학습 효과가 긍정적으로 확인되고 있으나, 대부분 **50명 이하 학급**에서의 성과임
- Behavior 모듈 과목(사회심리, 임상심리, 발달심리 등)은 수강 인원이 **60명 이상**인 경우가 다수이며, 이 경우 **12~15개 소그룹**이 동시에 운영되어야 함
- 메타분석 결과, 학급 규모가 50명을 초과하면 PBL 효과가 감소하는 경향이 있어 **TA 배치, 분반 운영, 수강 인원 조정** 등의 보완책이 필요함
- 본 토론은 대규모 수업에서 PBL이 불가능한지가 아니라, **어떤 조건이 갖추어져야 실현 가능한지**를 논의하는 자리임

## 그룹 C: Method + 전체 모듈

| | |
|---|---|
| **구성원** | 차지욱 (PI, Method 모듈 리드, N1-N4·E1·E9), Sowon Hahn, 이수현, 학과장 |
| **진행자** | 차지욱 |

### 토론 질문

> **"Method 모듈 4과목 신설로 학생 수강 부담이 과도하지 않은가?"**

### 배경 정보

- Method 모듈에 신설 예정인 4과목(연구입문, AI for Psychology, 데이터분석 I, 캡스톤)은 모두 **코딩 및 데이터 분석 역량**을 요구하며, R/Python 등의 프로그래밍이 수반됨
- 사회경제적 배경이 다양한 학생, 편입생, 유학생 등은 코딩 경험이 부족할 수 있어 **형평성(equity) 문제**가 발생할 수 있음
- 이를 완화하기 위한 방법이 존재함: GUI 기반 도구(PsychoPy Builder, JASP), tidyverse 기반 교수법, Use-Modify-Create 단계적 스캐폴딩 등
- 본 토론은 Method 과목 신설 자체의 필요성이 아니라, **학생 부담 수준의 적정성과 형평성 확보 방안**을 논의하는 자리임
