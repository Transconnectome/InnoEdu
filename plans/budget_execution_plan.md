## 5. 예산 집행 기준 및 유의사항

### 5.1 비목별 집행 기준

#### 인건비

| 기준 | 내용 |
|------|------|
| 지급 대상 | 대학원생 보조연구원(RA) 2명, 석사과정 이상 |
| 지급 기준 | 월 525,000원/인, 매월 25일 지급 |
| 채용 절차 | 사업책임자 추천 → 학과장 승인 → 채용 |
| 근무 시간 | 주 15시간 이내 (서울대학교 RA 운영 기준 준수) |
| 증빙 | 채용 계약서, 출근부, 업무 일지 |

#### 연구활동비

| 기준 | 내용 |
|------|------|
| 집행 절차 | 집행 계획서 작성 → 승인 → 집행 → 증빙 제출 |
| 승인 권한 | 500만원 미만: 사업책임자 승인, 500만원 이상: 심리학혁신위원회 승인(거버넌스 헌장 제10조) |
| 물품 구매 | 100만원 이상 3인 견적 비교, 50만원 이상 전자 결재 |
| 강사료 | 서울대학교 외부 강사 초빙 기준 적용 (시간당 단가 × 시수) |
| 증빙 | 세금계산서, 카드 매출전표, 이체 확인서 등 |

#### 연구수당

| 기준 | 내용 |
|------|------|
| 지급 대상 | 사업책임자 1명, 참여교수 11명, 학생 연구보조 |
| 지급 기준 | 분기별 정액 지급 (연구 기여도 확인 후) |
| 지급 조건 | 분기별 활동 보고서 제출, 위원회·FLC 참여 실적 확인 |
| 증빙 | 활동 보고서, 회의 참석 기록, 연구 산출물 목록 |

### 5.2 예비비 사용 조건

| 조건 | 내용 |
|------|------|
| 사용 범위 | 사전에 예측하지 못한 긴급 소요 또는 기존 항목의 추가 소요 |
| 승인 절차 | 사업책임자 요청 → 학과장 승인 (100만원 이상: 심리학혁신위원회 사전 보고) |
| 사용 제한 | 인건비 신규 채용 불가, 기존 비목 간 전용 우선 검토 후 집행 |
| 보고 의무 | 예비비 사용 시 차기 위원회 회의에서 사용 내역 보고 |
| 잔액 처리 | 잔액은 2차년도 이월 또는 기관 반납 (규정에 따름) |

**예비비 사용 예상 시나리오**:
- AI 도구 긴급 업그레이드 또는 신규 라이선스 추가 (최대 2,000,000원)
- 학술대회 추가 참가 또는 국제학술대회 지원 (최대 2,000,000원)
- PBL 시범 운영 중 추가 자료·장비 소요 (최대 1,500,000원)
- 기타 불가항력 소요 (최대 1,500,000원)

### 5.3 집행률 관리 목표

### 5.4 비목 간 전용 기준

| 전용 유형 | 조건 |
|----------|------|

| 비목 간 전용 | 전용 금액이 해당 비목의 20% 이내: 사업책임자 + 학과장 승인 |
| 비목 간 전용 (20% 초과) | 심리학혁신위원회 의결 + 연구처 사전 승인 필요 |
| 전용 금지 | 인건비 → 연구수당, 연구수당 → 인건비 전용 원칙적 금지 |

### 5.5 집행 유의사항

1. **선집행 금지**: 모든 예산은 사전 승인 후 집행하며, 소급 적용 불가
2. **증빙 관리**: 집행 후 5영업일 이내 증빙 서류를 사업팀에 제출
3. **회계 연도**: 2026년 3월 1일 ~ 12월 31일 (10개월)이며, 연도 말 잔액은 규정에 따라 처리
4. **감사 대비**: 월별 집행 내역을 정리하여 분기 리뷰 시 제출하며, 외부 감사에 대비
5. **분할 집행 금지**: 승인 절차 회피를 위한 건별 분할 집행 금지
6. **Bootcamp 정산**: 교수 Bootcamp(5,000,000원) 및 학부생 Bootcamp(3,000,000원)는 프로그램 종료 후 2주 이내 정산 완료
