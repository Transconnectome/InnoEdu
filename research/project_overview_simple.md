# InnoEdu 2031 — 프로젝트 진행 요약

> 이 문서는 InnoEdu 2031 프로젝트에서 수행한 문헌 연구와 교과과정 근거 매핑 작업을 비전문가도 이해할 수 있도록 정리한 것이다.
> 작성일: 2026-03-13

---

## 1. 이 프로젝트는 무엇인가

서울대학교 심리학과가 AI 시대에 맞게 교과과정을 혁신하려는 프로젝트이다. 핵심 질문은 이것이다:

> "심리학 교육을 어떻게 바꿔야 하는가? 그 근거는 무엇인가?"

감이나 경험이 아닌, **전 세계 학술 연구에 기반한 근거**를 확보하기 위해 대규모 문헌 조사(Scoping Review)를 수행했다.

---

## 2. 무엇을 했는가

전체 작업은 크게 세 단계로 나뉜다.

### 2.1 문헌 수집 — "어떤 연구가 있는가?"

전 세계 학술 논문, 교육 보고서, 대학 교육과정 사례 등 **600편**을 체계적으로 수집했다. 무작위로 모은 것이 아니라, 7개 주제 분야를 미리 설계하고 각 분야에 필요한 문헌을 찾았다.

| 주제 분야 | 다루는 내용 | 문헌 수 |
|----------|-----------|------:|
| 교과과정 설계 | 심리학 교과 구성, 뇌과학·인지·사회심리 교육 | ~100편 |
| AI와 데이터 교육 | AI 리터러시, Python/R 교육, 머신러닝 | ~100편 |
| PBL·교수법 | 문제기반학습, 능동학습, 플립드 클래스룸 | ~80편 |
| 학부생 연구역량 | 졸업논문, 연구윤리, 연구 멘토링 | ~70편 |
| 교수 역량 개발 | AI 교수법 훈련, 교수학습 공동체 | ~70편 |
| 글로벌 벤치마크 | Stanford, MIT, Harvard 등 해외 대학 사례 | ~80편 |
| 학습 측정·평가 | 역량 평가 도구, 학습분석, 루브릭 | ~100편 |

4차에 걸쳐 수집했다:
- **1차(P0)**: 핵심 60편으로 출발
- **2차(P1)**: 205편 확장
- **3차(P2)**: 165편 (한국어 문헌 36편 포함)
- **4차(P3)**: 170편 (빈 곳 채우기)

### 2.2 품질 검증 — "이 문헌들이 믿을 만한가?"

600편 전체를 대상으로 다음을 확인했다:

- **서지 검증**: 저자, 제목, 출판 정보가 실제로 맞는지 DOI 등을 통해 교차 확인
- **PRISMA-ScR**: 의학·사회과학에서 쓰는 국제 표준 보고 방식을 적용 (22개 체크리스트)
- **핵심 50편 선별**: 600편 중 가장 중요한 50편을 5가지 기준(교과 연계도, 인용 빈도, 학술지 등급, 방법론, 최신성)으로 뽑음
- **분야별 종합 분석**: 7개 분야별로 핵심 발견을 요약하는 종합 보고서 작성

### 2.3 교과과정 연계 — "이 연구가 어떤 수업에 적용되는가?"

600편의 연구를 서울대 심리학과의 **14개 과목**에 연결했다. 이것이 **Evidence Map(근거 매핑)**이다.

**새로 만드는 과목 4개:**

| 과목 | 학년 | 한마디 설명 |
|------|------|-----------|
| 심리학 연구 입문 (N1) | 1학년 | AI로 논문 찾기, 연구윤리, 데이터 읽기 |
| 심리과학을 위한 인공지능 (N2) | 2학년 | Python으로 AI·머신러닝 배우기 |
| 심리데이터 분석 I (N3) | 2학년 | R/Python으로 통계 분석하기 |
| 학부생 연구 캡스톤 (N4) | 4학년 | 졸업논문 프로젝트 |

**기존 과목 중 강화하는 과목 10개:**

| 과목 | 무엇이 바뀌는가 |
|------|--------------|
| 심리통계학 (E1) | 손계산 → R/Python 코딩 비중 40% |
| 실험심리입문 (E2) | 전통 실험 → PsychoPy 디지털 실험 |
| 인지과정 (E3) | 전통 인지 이론 → AI 인지 모델링 추가 |
| 임상신경심리 (E4) | 이론 위주 → fMRI/EEG 실습 추가 |
| 발달심리학 (E5) | 이론 위주 → 종단 데이터 시각화 |
| 사회심리학 (E6) | 전통 → SNS 감성분석 추가 |
| 조직심리학 (E7) | 이론 → HR 데이터 분석 추가 |
| 임상심리학 (E8) | 전통 → 디지털 치료·원격 심리 |
| 심리측정과 검사 (E9) | 지필 검사 → 디지털·컴퓨터 적응검사 |
| 뇌이미징 데이터사이언스 (E10) | 기초 분석 → 딥러닝 뇌영상 분석 |

각 과목에 대해 **"왜 이렇게 바꿔야 하는지"**를 학술 연구로 뒷받침했다. 예를 들어:

- "PBL(문제기반학습) 비중을 50%로 높여야 한다" ← Freeman et al. (2014): 능동학습이 성적을 0.47 표준편차 향상시킨다는 225개 연구 메타분석
- "AI 도구 사용 시 반드시 가이드라인을 제공해야 한다" ← Ma et al. (2025): 안내 있는 AI 사용이 안내 없는 사용보다 효과가 0.83 더 크다
- "팀 프로젝트는 4-5명이 최적" ← Zhang & Ma (2023): PBL 메타분석에서 4-5명 그룹이 가장 효과적

또한 미국심리학회(APA)의 학습 목표 5개 + 통합 주제 4개와 14개 과목이 어떻게 연결되는지도 매핑했다.

## 4. 중요한 판단 원칙

이 작업에서 지킨 핵심 원칙:

1. **근거 매핑은 "주제별"로 했다**: "몇 주차에 무엇을 가르쳐라"는 식으로 수업 일정을 정하지 않았다. 그것은 담당 교수가 결정할 사항이다. 이 매핑은 "이 주제를 가르칠 때 참고할 연구 근거"를 제공하는 것이다.

2. **기존 과목은 "변경 사항"에 초점을 맞췄다**: 이미 있는 과목(E1~E10)에 대해서는 과목 전체를 새로 설계하지 않고, 개편을 통해 새로 추가되는 부분의 학술적 근거를 제시했다.

3. **한국 맥락을 반영했다**: 해외 연구만이 아니라 한국어 문헌 54편, 한국 대학 사례를 포함하여 국내 적용 가능성을 확보했다.

4. **기관 문서 장르를 준수했다**: 모든 문서는 "서울대학교 심리학과는", "본 과목은" 같은 기관 주어를 사용하며, 개인적 의견("나는 생각한다")은 일절 포함하지 않았다.
