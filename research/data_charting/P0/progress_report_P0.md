## 3. 분기별 핵심 발견

### B1: BMBM 교과과정 설계 (12편)

**핵심 주제**:
1. **신경교육학/MBE 집중** (6편/50%): Brain 모듈 근거 기반 강력
2. **학제간 교과 설계** (3편): 모듈형 접근의 이론적 근거
3. **계산신경과학 교육** (2편): E3, E10 교과목 설계 참조
4. **APA 표준** (1편): 전체 교과 정렬 프레임워크

**강점**: Brain 모듈 근거 탄탄
**약점**: Mind, Behavior, Method 모듈 전용 소스 0편

### B2: AI·데이터 기반 연구 방법론 (10편)

**핵심 주제**:
1. **LLM/GenAI 심리학 활용** (4편): 행동 개입, 심리 평가, 연구, 임상
2. **AI 윤리·책임** (4편): 편향, 책임있는 개발, 거버넌스
3. **오픈사이언스·재현성** (3편): Nosek 재현위기 + 구조적 변화 + OSF
4. **통계/데이터 교육** (1편): R 기반 교수법

**강점**: LLM 활용과 AI 윤리 근거 풍부
**약점**: Python 교육, ML 응용, 데이터사이언스 교과 설계, 연구파이프라인 AI 소스 부족

### B3: PBL 및 수업방법 혁신 (9편)

**핵심 주제**:
1. **PBL 효과의 양적 근거**: Freeman(225개 연구, 0.47 SD, 55% 실패율 감소) + Zhang(SMD=0.441)
2. **PBL 설계 원리**: 유연한 지식, 메타인지, 내재적 동기, 실제적 맥락
3. **CURE 모델 확장성**: 사회과학에서도 효과적
4. **PBL-자기효능감**: 캡스톤 과목에서 유의미한 향상

**강점**: 메타분석 기반 강력한 효과 근거
**약점**: 심리학 특화 PBL 연구, Flipped Classroom, TBL, AI 교수법 소스 부재

### B4: 학부생 연구역량 파이프라인 (8편)

**핵심 주제**:
1. **URP = High-Impact Practice**: Lopatto(S35), CUR(S32) 확립
2. **멘토링이 핵심**: Linn(S36) — 성공의 최대 요인이나 근거 수준 약점 지적
3. **장벽 제거 전략**: Wayment(S37) 5가지 장벽-5가지 해결 → 참여율 3배 증가
4. **전 학부생 연구 경험 원칙**: Healey & Jenkins(S39) — 4년 파이프라인의 철학적 기반

**강점**: URP/CURE/HIP 이론적 기반 충실
**약점**: IRB 교육(B4.3), 학술 발표(B4.4), 심리학 특화 다중트랙 사례 부재

### B5: 교수역량 강화 (7편)

**핵심 주제**:
1. **AI Bootcamp 3대 모델**: Stanford(자기주도 모듈), Harvard(학과 방문 워크숍), MIT(Book Club + Practicum)
2. **FLC 이론**: Cox(2004) — 12명 이하, 학제간, 정기 모임 → TIP 설계 직접 근거
3. **SoTL 발전 경로**: Huber & Hutchings(2005) Teaching Commons → 교수 개발 3단계 최종 목표
4. **TPACK 프레임워크**: Mishra & Koehler(2006) — 교수 AI 리터러시 진단 틀

**강점**: 글로벌 Bootcamp 모델 3개 확보
**약점**: 성과 평가 데이터 없음(기술적 기술만), CTL 비교 연구 없음, Tier 1 비율 매우 낮음(18.8%)

### B6: 글로벌 벤치마크 및 정책 (9편)

**핵심 주제**:
1. **5개 대학 차별화 전략**: Stanford(기관 정책), Harvard(AI 정책 프레임워크), MIT(AI 리터러시), Oxford(계산심리학), NUS(GenAI 실전)
2. **APA 3.0 정렬**: BMBM 최상위 정렬 기준
3. **한국 교육부**: AI 인재양성 예산 25배 확대 — 국가 정책 정합성 근거
4. **"실험 + 가드레일" 합의**: AI 전면 금지보다 과목별 맞춤형 정책 지향

**강점**: Stanford/Harvard 잘 문서화
**약점**: Oxford/NUS 피상적, 아시아 대학 사례 부족, 서울대 자체 사례 없음

### B7: 측정·평가·거버넌스 (5편)

**핵심 주제**:
1. **4대 평가 도구**: SRA(통계추론), SOLO(학습구조), AI Literacy(AI역량), K-DOCS(창의성)
2. **프레임워크 vs 표준화 도구 구분**: 평가 체계 설계 시 차이 고려 필요
3. **거버넌스 초기 단계**: UNESCO/OECD 수준의 비구속적 지침
4. **교차문화 적응 필수**: 모든 도구 영어권 개발 → 한국어 적응 전제

**강점**: 핵심 측정 도구 4개 비교표 완성, Tier 1 80%
**약점**: B7.2 프로그램 평가(0편), B7.4 AI 정책·학술 무결성(0편) — **최우선 보강 필요**

**참고**: S59 "CREWS Creativity Rubric" 명칭 검증 불가 → Kaufman K-DOCS/CAT로 대체 차팅, P1에서 재확인 필요.
