## 3. 핵심 교육 주제-근거 연계

### 3.1 R/Python 프로그래밍 환경

**교육 목표**: R/RStudio 환경을 설정하고 기본 문법을 습득하며, Python(Pandas, Seaborn) 기초 역량을 확보하여 두 언어 간 전환 능력을 형성한다.

**핵심 근거**:
- Glasgow PsyTeachR (2024): 학부-대학원 전 수준 R 기반 재현 가능 연구 교육 전환 완료 사례. Level 1-3 체계적 경로 → R 환경 설정 및 기초 문법 교육의 글로벌 벤치마크
- Navarro (2022) Learning Stats with JASP/R: CC BY-SA 4.0 오픈 교재 → R 기초 교육의 핵심 참조 교재
- P1-B2-019: learnr/Shiny 기반 인터랙티브 OER — 사회과학 학부생의 제한적 수학/통계 배경 고려 → 비전공자 접근성 확보 전략
- Dalmaijer et al. (2024, P1-B2-016): Python for Experimental Psychologists 2판 — Pandas/Statsmodels 활용 분석 → 심리학 맥락 Python 교육의 직접 참조
- UT Austin (P3-S021): Behavioral Data Science Initiative — Python 기반 행동 데이터 분석 → 교과 벤치마크
- P1-B2-020: Washington 대학 증거 기반 Python 교수법 — 플립드 구조, 다중 표현 → Python 교수법 설계
- Mahner et al. (2025, P2-B2-015): 코딩의 10대 원칙(변경 추적, 코드 테스트, 코드 리뷰) → 코딩 실천 원칙의 학부 수준 도입. R에서 Python으로의 전환 시 일관된 코딩 실천 유지

**교수법 시사점**: R 초기 학습에는 learnr/Shiny 기반 인터랙티브 환경을 활용하여 비전공자 접근성을 확보한 후 독립 코딩으로 전환. Python 도입 시에는 플립드 구조(R→Python 전환 워크숍)를 적용.

**평가 시사점**: R 기초 문법 코딩 과제; Python 데이터분석 과제(Pandas 활용).

- Garrison (2024, P3-S020): DataScience4Psych R/tidyverse 교재 및 강좌 → 데이터 정제 워크플로 설계
- Cetinkaya-Rundel & Hardin (2023, S19): IMS 2e — tidyverse 기반 교육 + 32개 대화형 튜토리얼 → 대화형 실습 설계 참조. 시뮬레이션 기반 추론 접근에서 시각화 중심 EDA 강조
- Mahner et al. (2025, P2-B2-015): 코딩의 10대 원칙(변경 추적, 코드 테스트, 코드 리뷰) → 코딩 실천 원칙의 학부 수준 도입

- P3-S019 Duke 인지과학 시각화 프로젝트: 인지과학 원리 기반 데이터 시각화 → 심리학 맥락 시각화 프로젝트 참조

**평가 시사점**: 데이터 정제 과제(심리학 데이터셋 활용); EDA 보고서(ggplot2 시각화 포함).

### 3.3 추론 통계: t-test, ANOVA, 회귀분석

**교육 목표**: E1(심리통계학)에서 학습한 추론 통계 개념을 R 코딩 기반으로 실습하고, 효과크기 보고와 가정 검증을 포함한 분석 역량을 구축한다.

**핵심 근거**:
- Field et al. (2025): JASP/R 기반 통계 교재 — 빈도주의-베이지안 이중 접근을 학부 수준에서 구현 → t-test/ANOVA/회귀분석의 R 기반 실습 근거. 효과크기 보고와 가정 검증 강조
- Playfoot (2023): 심리학 학부생 통계 모듈 플립드 클래스룸 — 플립드 학생이 전통 강의 대비 유의미하게 높은 성적 → 플립드 구조 도입 근거
- UMN (P1-B2-018): 학부 통계에 R 의무화 — 통계 태도와 오픈사이언스 태도 간 강한 상관 → R 기반 통계 실습의 태도 변화 효과
- Navarro (2022): 회귀 챕터 — CC 교재로 한국어 번역/적용 가능 → 보조 교재 활용
- P1-B7-016: 통계 교육에서 PBL과 R 프로그래밍 도구의 통합이 학습 성과에 긍정적 영향 → R 기반 통계 실습의 PBL 통합 근거

**교수법 시사점**: 플립드 클래스룸(동영상 사전 학습 + 실습)을 핵심 구조로 적용. 심리학 데이터를 활용한 코딩 실습과 R Markdown 기반 보고서 작성을 결합.

**평가 시사점**: R 기반 t-test/ANOVA 분석 과제; 회귀분석 보고서(R Markdown 형식).

### 3.5 재현가능 연구와 R Markdown

**교육 목표**: 재현가능 연구 워크플로(사전등록, 데이터 공유, R Markdown 보고서)를 이해하고 직접 실천한다.

**핵심 근거**:
- Nosek et al. (2015): 100개 재현 연구 중 36% 재현 성공 — 재현성 위기의 규모를 확립하여 재현가능 워크플로우 교육의 당위성 제공
- Korbmacher et al. (2023): '위기'를 '혁명'으로 재프레이밍 — 구조적·절차적·공동체 3차원 긍정 변화 → 재현가능 연구를 교과과정의 핵심 원칙으로 채택
- Frank et al. (2025) Experimentology: 사전등록-데이터 공유-재현 가능 보고서의 통합 워크플로 모델. R Markdown/Quarto 기반 재현가능 보고서를 연구 워크플로 표준으로 제시
- Beaudry et al. (2024, P2-B2-017): 신입생 18%만 "재현성 위기" 인지 → 2학년 시점에서 재현성 인식 교육의 필요성
- Pownall et al. (2023, P2-B2-016): 사전등록 집단이 오픈사이언스 이해도 유의미 향상 → 사전등록과 재현가능 보고서의 연계 교육
- Glasgow PsyTeachR (2024): Level 2-3 — R Markdown 기반 보고서를 필수 산출물로 설정 → 교육 경로 참조
- P2-B2-025: 사전등록 연구가 방법론적 엄밀성·투명성에서 우위 확인 → 사전등록 통합의 실증적 근거

**교수법 시사점**: 강의(재현 실패 사례 제시)와 R Markdown 보고서 작성 실습을 결합. 재현가능 연구 원칙을 과목 전반의 핵심 원칙으로 관통시키고, R Markdown 기반 보고서를 필수 산출물로 설정.

**평가 시사점**: 재현성 인식 사전-사후 설문; R Markdown 보고서 산출물 평가.

### 3.6 오픈 데이터셋 활용과 오픈사이언스 도구

- Markant & Galati (2023): OSCI — 오픈사이언스 개념 목록 → 오픈사이언스 이해도 평가 도구

## 7. 구현 권고사항

### 우선 도입 요소 (1차년도)

1. **R/tidyverse 기반 교육 체계 구축**: Glasgow PsyTeachR와 ds4psy 모델을 참조하여 R/RStudio 기반 실습 환경 구축. Navarro (2022) CC 교재를 한국어 번역/심리학 데이터 적용하여 보조 교재로 활용.
2. **재현가능 워크플로 필수화**: Frank et al. (2025)의 Experimentology 모델에 따라 R Markdown 기반 보고서를 필수 산출물로 설정. R Markdown 집중 교육 포함.
3. **오픈사이언스 실습 통합**: OSF 프로젝트 생성 및 사전등록 초안 작성 실습. Nosek et al. (2015)과 Korbmacher et al. (2023)의 재현성 혁명 맥락에서 교육.
4. **PBL 4-5명 팀 구성**: Zhang & Ma (2023)의 메타분석 근거에 따른 최적 그룹 크기 설정. SOLO 기반 형성적 평가를 교수자 주도로 구조화.
5. **심리학 공개 데이터셋 확보**: 실습용 공개 데이터셋 확보 — ABCD Study (Saragosa-Harris et al. 2022), OSF 공개 데이터, 한국아동청소년패널 등.

### 점진적 도입 요소 (2차년도)

1. **AI 튜터 시범 도입**: Ma et al. (2014, 2025)의 메타분석 근거에 기반하여 코딩 오류 피드백 및 통계 개념 설명을 위한 AI 튜터 시범 운영. AI 거버넌스 태스크포스 감독 하 운영.
2. **Python 비중 확대**: 1차년도에는 입문 수준이나, 2차년도에는 R-Python 병행 비중을 3:2로 확대 검토. Dalmaijer et al. (2024) 교재 활용.
3. **ESM/디지털 표현형 데이터 모듈 추가**: P1-B2-025(ESM-Quest)와 P3-S016(웨어러블/패시브 센싱) 근거에 따라 디지털 표현형 데이터 수집-분석 실습 모듈을 오픈 데이터셋 활용 주제에 통합.
4. **NLP 기반 텍스트 분석 입문**: B2 발견 6에서 확인된 ML/NLP 확산 추세에 따라, 텍스트 데이터 분석 기초 모듈 추가. P1-B2-013(Nature Reviews Psychology NLP 리뷰) 참조.
