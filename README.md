## 핵심 전략: BMBM 프레임워크

전공 39개 과목을 4대 모듈로 체계적 재편:

| 모듈 | 영역 | 과목 수 | 비중 |
|------|------|---------|------|
| **Brain** (뇌) | 인지신경과학, 뇌영상, 계산신경과학 | 11개 | 28% |
| **Mind** (마음) | 인지심리, 지각심리, 언어심리 | 8개 | 20.5% |
| **Behavior** (행동) | 사회, 임상, 발달, 조직심리 | 12개 | 31% |
| **Method** (방법) | 통계, AI, 데이터분석, 연구방법 | 8개 | 20.5% |

### 교과과정 개편 (14개, 41%)

**신설 4개** (모두 Method 모듈):

| 과목 | 학년 | 학점 | 핵심 내용 |
|------|------|------|----------|
| N1: 심리학 연구 입문 | 1학년 | 2 | AI 문헌검색, 연구윤리, 데이터 리터러시 |
| N2: 심리과학을 위한 인공지능 | 2학년 | 3 | Python ML/NLP, 영역별 AI 활용 |
| N3: 심리데이터 분석 I | 2학년 | 3 | R/Python 데이터분석, 재현가능 워크플로우 |
| N4: 학부생 연구 캡스톤 | 4학년 | 3 | 전체 연구 파이프라인, 3트랙 졸업논문 |

**강화 개편 10개**: E1(심리통계학), E2(실험심리입문), E3(인지과정), E4(임상신경심리), E5(발달심리), E6(사회심리), E7(조직심리), E8(임상심리), E9(심리측정), E10(뇌이미징 DS)
