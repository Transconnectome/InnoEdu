## 4대 성과지표

### 1. 교과과정 개편
- 14개 교과 신설/개편 (전체 41%)
  - **신설 4개**: 심리학 연구 입문, 심리과학을 위한 인공지능, 심리데이터 분석 I, 학부생 연구 캡스톤
  - **강화 개편 10개**: 기존 과목에 AI·데이터 분석 실습 추가
- 모듈형 이수체계 도입

### 2. 수업방법 혁신 및 개선
- Project Based Learning 비중 확대: 현행 13% → 목표 50%
- Brain-Mind-Behavior Studio 실습 공간 구축
- 수업 구성: 강의 40% + 실습 40% + 연구 프로젝트 20%

### 3. AI·데이터 기반 심리학 연구역량 체계 구축
- 학부생-교수 연구 매칭 지원 체계 설계
- 연구윤리 교육 및 IRB 프로세스 교육 프로그램 개발
- Python/R 기반 데이터 분석 실습

### 4. 학부생 연구 역량 강화
- 졸업논문 제도 다양화: 연구 중심, 실무 중심, 포트폴리오 트랙
- 10주 연구 지도 프로그램 (문헌검토→설계→수행→분석→작성)
- 학술대회 참가 및 논문 투고 지원 체계

## 교수-학생 동반 혁신 생태계

### 교원 역량 강화 3단계
1. **AI Teaching Bootcamp** (5일): Claude Code, ChatGPT, Gemini 기초 실습 + 영역별 AI 활용 사례 Workshop
2. **AI 교수법 집중 워크숍**: 7개 영역별 소그룹, AI 활용 수업 모델 개발 및 공유
3. **월간 AI 교수법 클리닉**: 우수 사례 공유 + 실습 + Q&A

### 창의적 학습 환경
- AI 튜터·LA(Learning Assistant) 제도 도입
- 몰입도 측정(CORE-Q)과 창의성 진단
- 동문·산업 전문가 멘토 30명 연계
- Reflective Studio: 감정·몰입 저널 작성을 통한 자기 성찰

### 서울대 심리학과 차별화 포인트
- **교육 초점**: AI 도구 활용이 아닌 심리학 연구 방법론과 AI의 유기적 통합
- **연구 파이프라인**: 문헌분석 → 데이터수집 → AI·통계분석 → 논문작성/모델개발
- **교수 역량 강화**: 예산의 20% 투자 (타 대학 대비 2-4배)
- **측정 과학성**: SRA, SOLO Taxonomy, AI Literacy Scale 등 검증된 도구

## APA Guidelines 3.0 연계

본 교육 혁신 계획은 APA 5대 학습목표를 체계적으로 연계:

| APA 학습목표 | AI·데이터 기반 연구 방법론 통합 |
|------------|------------------------|
| Goal 1: 지식 기반 | 체계적 문헌 분석, 연구 GAP 식별 |
| Goal 2: 과학적 탐구 | 실증 데이터 수집 및 통계 분석 |
| Goal 3: 윤리적 가치관 | 연구 윤리 및 데이터 보호 |
| Goal 4: 의사소통 | 연구 논문 작성 및 학술 발표 |
| Goal 5: 전문성 개발 | 연구자 정체성 형성 및 진로 개발 |

4대 통합 주제(Cross-Cutting Themes): Civic Engagement, Sustainability, Equity & Justice, Digital Fluency
